creakingdoor-rpy
================

RPi Halloween creakingdoor script using Rpi + mpg123 + python + sensor

##### Requires
* [mpg123](http://www.mpg123.de/)

## Installing ``mpg123``
##### In order to install using OSX/HomeBrew:

``brew install mpg123``


##### To install using Debian and sons just type:

``sudo apt-get install mpg123``
